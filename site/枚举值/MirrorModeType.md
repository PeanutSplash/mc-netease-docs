#  MirrorModeType

class in mod.common.minecraftEnum

*   描述
    
    描述放置结构体时的镜像模式
    

```
class MirrorModeType:
	NONE = 0  		# 不使用镜像
	X = 1  			# 沿X轴进行镜像放置
	Y = 2  			# 沿Y轴进行镜像放置
	XY = 3  		# 沿XY轴进行镜像放置


```

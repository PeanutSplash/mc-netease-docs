#  GameType

class in mod.common.minecraftEnum

*   描述
    
    描述游戏类型的枚举值
    

```
class GameType(object):
	Undefined = -1            # 未定义类型
	Survival = 0              # 生存模式
	Creative = 1              # 创造模式
	Adventure = 2             # 冒险模式
	Spectator = 6             # 旁观模式
	Default = Survival        # 默认类型，默认为生存模式


```

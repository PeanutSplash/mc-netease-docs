#  AnimationModeType

class in mod.common.minecraftEnum

*   描述
    
    描述放置结构体时的动画模式
    

```
class AnimationModeType:
	NONE = 0  		# 无动画模式
	LAYERS = 1  	# 按照层数放置
	BLOCKS = 2  	# 按照方块放置


```

##  世界

事件

描述

[AchievementCompleteEvent](/事件/世界)

服务端

玩家完成自定义成就时触发该事件

[AddEntityClientEvent](/事件/世界)

客户端

客户端侧创建新实体时触发

[AddEntityServerEvent](/事件/世界)

服务端

服务端侧创建新实体，或实体从存档加载时触发

[AddPlayerAOIClientEvent](/事件/世界)

客户端

玩家加入游戏时视野内存在其他玩家，或者其余玩家进入当前玩家视野时触发的事件，替换AddPlayerEvent

[AddPlayerCreatedClientEvent](/事件/世界)

客户端

玩家进入当前玩家所在的区块AOI后，玩家皮肤数据异步加载完成后触发的事件

[AddServerPlayerEvent](/事件/世界)

服务端

触发时机：玩家加入时触发该事件。

[ChunkAcquireDiscardedClientEvent](/事件/世界)

客户端

触发时机：客户端区块即将被卸载时

[ChunkAcquireDiscardedServerEvent](/事件/世界)

服务端

服务端区块即将被卸载时触发

[ChunkGeneratedServerEvent](/事件/世界)

服务端

触发时机：区块创建完成时触发

[ChunkLoadedClientEvent](/事件/世界)

客户端

触发时机：客户端区块加载完成时

[ChunkLoadedServerEvent](/事件/世界)

服务端

触发时机：服务端区块加载完成时

[ClientLoadAddonsFinishServerEvent](/事件/世界)

服务端

触发时机：客户端mod加载完成时，服务端触发此事件。服务器可以使用此事件，往客户端发送数据给其初始化。

[CommandEvent](/事件/世界)

服务端

玩家请求执行指令时触发

[CustomCommandTriggerServerEvent](/事件/世界)

服务端

自定义命令触发事件

[DelServerPlayerEvent](/事件/世界)

服务端

触发时机：删除玩家时触发该事件。

[EntityRemoveEvent](/事件/世界)

服务端

实体被删除时触发

[ExplosionServerEvent](/事件/世界)

服务端

当发生爆炸时触发。

[GameRenderTickEvent](/事件/世界)

客户端

客户端渲染帧开始时触发该事件,一秒触发次数为当前的帧数

[GlobalCommandServerEvent](/事件/世界)

服务端

服务端全局命令事件，包括聊天栏发送、SetCommand接口、命令方块（矿车）、行为包动画执行命令

[LoadClientAddonScriptsAfter](/事件/世界)

客户端

客户端加载mod完成事件

[LoadServerAddonScriptsAfter](/事件/世界)

服务端

服务器加载完mod时触发

[NewOnEntityAreaEvent](/事件/世界)

服务端

触发时机：通过RegisterEntityAOIEvent注册过AOI事件后，当有实体进入或离开注册感应区域时触发该事件。

[OnCommandOutputClientEvent](/事件/世界)

客户端

当command命令有成功消息输出时触发

[OnCommandOutputServerEvent](/事件/世界)

服务端

Command命令执行成功事件

[OnContainerFillLoottableServerEvent](/事件/世界)

服务端

触发时机：随机奖励箱第一次打开根据loottable生成物品时

[OnLightningLevelChangeServerEvent](/事件/世界)

服务端

打雷强度发生改变

[OnLocalLightningLevelChangeServerEvent](/事件/世界)

服务端

独立维度天气打雷强度发生改变时触发

[OnLocalPlayerStopLoading](/事件/世界)

客户端

触发时机：玩家进入存档，出生点地形加载完成时触发。该事件触发时可以进行切换维度的操作。

[OnLocalRainLevelChangeServerEvent](/事件/世界)

服务端

独立维度天气下雨强度发生改变时触发

[OnRainLevelChangeServerEvent](/事件/世界)

服务端

下雨强度发生改变

[OnScriptTickClient](/事件/世界)

客户端

客户端tick事件,1秒30次

[OnScriptTickServer](/事件/世界)

服务端

服务器tick时触发,1秒有30个tick

[PlaceNeteaseLargeFeatureServerEvent](/事件/世界)

服务端

触发时机：网易版大型结构即将生成时服务端抛出该事件。

[PlaceNeteaseStructureFeatureEvent](/事件/世界)

服务端

触发时机：首次生成地形时，结构特征即将生成时服务端抛出该事件。

[PlayerIntendLeaveServerEvent](/事件/世界)

服务端

触发时机：即将删除玩家时触发该事件，此时可以通过各种API获取玩家的当前状态。

[PlayerJoinMessageEvent](/事件/世界)

服务端

触发时机：准备显示“xxx加入游戏”的玩家登录提示文字时服务端抛出的事件。

[PlayerLeftMessageServerEvent](/事件/世界)

服务端

触发时机：准备显示“xxx离开游戏”的玩家离开提示文字时服务端抛出的事件。

[RemoveEntityClientEvent](/事件/世界)

客户端

客户端侧实体被移除时触发

[RemovePlayerAOIClientEvent](/事件/世界)

客户端

玩家离开当前玩家视野时触发的事件

[ServerChatEvent](/事件/世界)

服务端

玩家发送聊天信息时触发

[ServerPostBlockPatternEvent](/事件/世界)

服务端

触发时机：用方块组合生成生物，生成生物之后触发该事件。

[ServerPreBlockPatternEvent](/事件/世界)

服务端

触发时机：用方块组合生成生物，在放置最后一个组成方块时触发该事件。

[ServerSpawnMobEvent](/事件/世界)

服务端

游戏内自动生成生物，以及使用api生成生物时触发

[UnLoadClientAddonScriptsBefore](/事件/世界)

客户端

客户端卸载mod之前触发

##  实体

事件

描述

[ActorHurtServerEvent](/事件/实体)

服务端

触发时机：生物（包括玩家）受伤时触发

[ActuallyHurtServerEvent](/事件/实体)

服务端

实体实际受到伤害时触发，相比于DamageEvent，该伤害为经过护甲及buff计算后，实际的扣血量

[AddEffectServerEvent](/事件/实体)

服务端

触发时机：实体获得状态效果时

[ApproachEntityClientEvent](/事件/实体)

客户端

玩家靠近生物时触发

[ChangeSwimStateServerEvent](/事件/实体)

服务端

触发时机：实体开始或者结束游泳时

[DamageEvent](/事件/实体)

服务端

实体受到伤害时触发

[EntityChangeDimensionServerEvent](/事件/实体)

服务端

实体维度改变时服务端抛出

[EntityDefinitionsEventServerEvent](/事件/实体)

服务端

触发时机：生物定义json文件中设置的event触发时同时触发。生物行为变更事件

[EntityDieLoottableAfterServerEvent](/事件/实体)

服务端

触发时机：生物死亡掉落物品之后

[EntityDieLoottableServerEvent](/事件/实体)

服务端

触发时机：生物死亡掉落物品时

[EntityDroppedItemServerEvent](/事件/实体)

服务端

触发时机：生物扔出物品时触发

[EntityEffectDamageServerEvent](/事件/实体)

服务端

[EntityLoadScriptEvent](/事件/实体)

服务端

[EntityModelChangedClientEvent](/事件/实体)

客户端

触发时机：实体模型切换时触发

[EntityMotionStartServerEvent](/事件/实体)

服务端

实体运动器开始事件。实体（包含玩家）添加运动器后，运动器开始运行时触发

[EntityMotionStopServerEvent](/事件/实体)

服务端

实体运动器停止事件。实体（包含玩家）添加运动器并开始运行后，运动器自动停止时触发

[EntityPickupItemServerEvent](/事件/实体)

服务端

有minecraft:behavior.pickup\_items行为的生物拾取物品时触发该事件，例如村民拾取面包、猪灵拾取金锭

[EntityStartRidingEvent](/事件/实体)

服务端

[EntityStopRidingEvent](/事件/实体)

服务端

触发时机：当实体停止骑乘时

[EntityStopRidingEvent](/事件/实体)

客户端

触发时机：当实体停止骑乘时

[EntityTickServerEvent](/事件/实体)

服务端

实体tick时触发。该事件为20帧每秒。需要使用AddEntityTickEventWhiteList添加触发该事件的实体类型白名单。

[HealthChangeBeforeServerEvent](/事件/实体)

服务端

[HealthChangeClientEvent](/事件/实体)

[HealthChangeServerEvent](/事件/实体)

服务端

生物生命值发生变化时触发

[LeaveEntityClientEvent](/事件/实体)

[MobDieEvent](/事件/实体)

服务端

实体死亡时触发

[MobGriefingBlockServerEvent](/事件/实体)

服务端

环境生物改变方块时触发，触发的时机与mobgriefing游戏规则影响的行为相同

[OnFireHurtEvent](/事件/实体)

服务端

生物受到火焰伤害时触发

[OnGroundClientEvent](/事件/实体)

客户端

实体着地事件。玩家，沙子，铁砧，掉落的物品，点燃的TNT掉落地面时触发，其余实体着地不触发。

[OnGroundServerEvent](/事件/实体)

服务端

实体着地事件。实体，掉落的物品，点燃的TNT掉落地面时触发

[OnKnockBackServerEvent](/事件/实体)

服务端

实体被击退时触发

[OnMobHitBlockServerEvent](/事件/实体)

服务端

触发时机：通过OpenMobHitBlockDetection打开方块碰撞检测后，当生物（不包括玩家）碰撞到方块时触发该事件。

[OnMobHitMobClientEvent](/事件/实体)

客户端

触发时机：通过OpenPlayerHitMobDetection打开生物碰撞检测后，当生物间（包含玩家）碰撞时触发该事件。注：客户端和服务端分别作碰撞检测，可能两个事件返回的略有差异。

[OnMobHitMobServerEvent](/事件/实体)

服务端

触发时机：通过OpenPlayerHitMobDetection打开生物碰撞检测后，当生物间（包含玩家）碰撞时触发该事件。注：客户端和服务端分别作碰撞检测，可能两个事件返回的略有差异。

[ProjectileCritHitEvent](/事件/实体)

服务端

触发时机：当抛射物与头部碰撞时触发该事件。注：需调用OpenPlayerCritBox开启玩家爆头后才能触发。

[ProjectileDoHitEffectEvent](/事件/实体)

服务端

触发时机：当抛射物碰撞时触发该事件

[RefreshEffectServerEvent](/事件/实体)

服务端

触发时机：实体身上状态效果更新时触发，更新条件1、新增状态等级较高，更新状态等级及时间；2、新增状态等级不变，时间较长，更新状态持续时间

[RemoveEffectServerEvent](/事件/实体)

服务端

触发时机：实体身上状态效果被移除时

[SpawnProjectileServerEvent](/事件/实体)

服务端

触发时机：抛射物生成时触发

[StartRidingClientEvent](/事件/实体)

客户端

触发时机：一个实体即将骑乘另外一个实体

[StartRidingServerEvent](/事件/实体)

服务端

触发时机：一个实体即将骑乘另外一个实体

[WillAddEffectServerEvent](/事件/实体)

服务端

触发时机：实体即将获得状态效果前

[WillTeleportToServerEvent](/事件/实体)

服务端

实体即将传送或切换维度

##  玩家

事件

描述

[AddExpEvent](/事件/玩家)

服务端

触发时机：当玩家增加经验时触发该事件。

[AddLevelEvent](/事件/玩家)

服务端

触发时机：当玩家升级时触发该事件。

[CameraMotionStartClientEvent](/事件/玩家)

客户端

相机运动器开始事件。相机添加运动器后，运动器开始运行时触发

[CameraMotionStopClientEvent](/事件/玩家)

客户端

相机运动器停止事件。相机添加运动器并开始运行后，运动器自动停止时触发

[ChangeLevelUpCostServerEvent](/事件/玩家)

服务端

触发时机：获取玩家下一个等级升级经验时，用于重载玩家的升级经验，每个等级在重置之前都只会触发一次

[DimensionChangeClientEvent](/事件/玩家)

客户端

玩家维度改变时客户端抛出

[DimensionChangeFinishClientEvent](/事件/玩家)

客户端

玩家维度改变完成后客户端抛出

[DimensionChangeFinishServerEvent](/事件/玩家)

服务端

玩家维度改变完成后服务端抛出

[DimensionChangeServerEvent](/事件/玩家)

服务端

玩家维度改变时服务端抛出

[ExtinguishFireClientEvent](/事件/玩家)

客户端

玩家扑灭火焰时触发。下雨，倒水等方式熄灭火焰不会触发。

[ExtinguishFireServerEvent](/事件/玩家)

服务端

玩家扑灭火焰时触发。下雨，倒水等方式熄灭火焰不会触发。

[GameTypeChangedClientEvent](/事件/玩家)

客户端

个人游戏模式发生变化时客户端触发。

[GameTypeChangedServerEvent](/事件/玩家)

服务端

当默认游戏模式或个人游戏模式发生变化时服务端触发，如果个人游戏模式不为默认时，修改默认游戏模式也会同时修改个人游戏模式，此时会触发两次该事件

[MountTamingEvent](/事件/玩家)

服务端

玩家通过骑乘驯服生物后触发该事件

[OnLocalPlayerActionClientEvent](/事件/玩家)

客户端

玩家动作事件，当本地玩家开始/停止某些动作时触发该事件

[OnLocalPlayerStartJumpClientEvent](/事件/玩家)

客户端

本地玩家开始跳跃时触发

[OnPlayerActionServerEvent](/事件/玩家)

服务端

玩家动作事件，当玩家开始/停止某些动作时触发该事件

[OnPlayerHitBlockClientEvent](/事件/玩家)

客户端

触发时机：通过OpenPlayerHitBlockDetection打开方块碰撞检测后，当玩家碰撞到方块时触发该事件。玩家着地时会触发OnGroundClientEvent，而不是该事件。客户端和服务端分别作碰撞检测，可能两个事件返回的结果略有差异。

[OnPlayerHitBlockServerEvent](/事件/玩家)

服务端

触发时机：通过OpenPlayerHitBlockDetection打开方块碰撞检测后，当玩家碰撞到方块时触发该事件。监听玩家着地请使用客户端的OnGroundClientEvent。客户端和服务端分别作碰撞检测，可能两个事件返回的略有差异。

[PerspChangeClientEvent](/事件/玩家)

客户端

视角切换时会触发的事件

[PlayerAttackEntityEvent](/事件/玩家)

服务端

触发时机：当玩家攻击时触发该事件。

[PlayerAttackEntityEvent](/事件/玩家)

客户端

触发时机：当本地玩家攻击时触发该事件。

[PlayerCheatSpinAttackServerEvent](/事件/玩家)

Apollo

触发时机：玩家开始/结束快速旋转攻击并且不符合发送快速旋转攻击条件时触发（装备激流附魔的三叉戟、在水中或雨中，且未骑乘）

[PlayerDieEvent](/事件/玩家)

服务端

触发时机：当玩家死亡时触发该事件。

[PlayerDoInteractServerEvent](/事件/玩家)

服务端

玩家与有minecraft:interact组件的生物交互时触发该事件，例如玩家手持空桶对牛挤奶、玩家手持打火石点燃苦力怕

[PlayerEatFoodServerEvent](/事件/玩家)

服务端

触发时机：玩家吃下食物时触发

[PlayerFeedEntityServerEvent](/事件/玩家)

服务端

玩家喂养生物时触发，例如玩家手持小麦喂养牛、玩家手持胡萝卜喂养幼年猪。

[PlayerHungerChangeServerEvent](/事件/玩家)

服务端

玩家饥饿度变化时触发该事件

[PlayerHurtEvent](/事件/玩家)

服务端

触发时机：当玩家受伤害前触发该事件。

[PlayerInteractServerEvent](/事件/玩家)

服务端

触发时机：玩家可以与实体交互时。如果是鼠标控制模式，则当准心对着实体时触发。如果是触屏模式，则触发时机与屏幕下方的交互按钮显示的时机相同。玩家真正与实体发生交互的事件见[PlayerDoInteractServerEvent](/事件/玩家)

[PlayerNamedEntityServerEvent](/事件/玩家)

服务端

玩家用命名牌重命名实体时触发，例如玩家手持命名牌对羊修改名字、玩家手持命名牌对盔甲架修改名字。

[PlayerPermissionChangeClientEvent](/事件/玩家)

客户端

玩家权限变更事件

[PlayerPermissionChangeServerEvent](/事件/玩家)

服务端

玩家权限变更事件

[PlayerRespawnEvent](/事件/玩家)

服务端

触发时机：玩家复活时触发该事件。

[PlayerRespawnFinishServerEvent](/事件/玩家)

[PlayerSleepServerEvent](/事件/玩家)

服务端

玩家使用床睡觉成功

[PlayerSpinAttackServerEvent](/事件/玩家)

Apollo

触发时机：玩家开始/结束快速旋转攻击时触发

[PlayerStopSleepServerEvent](/事件/玩家)

服务端

玩家停止睡觉

[PlayerTeleportEvent](/事件/玩家)
